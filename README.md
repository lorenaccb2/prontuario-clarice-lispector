# Prontuário Clarice Lispector

## 🧠 Visão Geral

Este Repositório é referente ao projeto de Gestão de Processos de Negócio, disciplina do 5° período do Centro de Informática da UFPE. O objetivo é melhorar a experiência de uso do sistema do prontuário Clarice Lispector utilizado no atendimento de mulheres, tornando-o mais intuitivo, eficiente e centrado nas necessidades dos profissionais envolvidos no processo.

A proposta foca na criação de uma **nova camada de interface**, capaz de unificar o fluxo de atendimento e reduzir a complexidade operacional.

---

## 🎯 Objetivo do Projeto

Projetar uma nova interface para o sistema **Clarice Lispector**, com foco em:

- Unificar as etapas do fluxo de atendimento  
- Reduzir a complexidade cognitiva dos usuários  
- Melhorar a visualização das informações  
- Apoiar a tomada de decisão durante o atendimento  

---

## 🧩 Contexto

O sistema é utilizado em um ambiente de atendimento multidisciplinar, envolvendo diferentes áreas:

- Recepção  
- Serviço Social  
- Psicologia  
- Jurídico  
- Gestão do centro  

---

## ⚠️ Problemas Identificados

- Interface confusa e pouco intuitiva  
- Excesso de informações, dificultando a busca de dados relevantes  
- Fluxo fragmentado entre várias telas e etapas  
- Sistema focado em armazenar dados, não em apoiar o usuário  
- Uso de ferramentas externas (como n8n) para contornar limitações  
- Múltiplas ações acontecendo simultaneamente, aumentando a complexidade  

---

## 💡 Proposta de Valor

A solução proposta busca:

✔️ Melhorar a usabilidade do sistema  
✔️ Reduzir erros operacionais  
✔️ Aumentar a produtividade das equipes  
✔️ Centralizar informações relevantes  
✔️ Apoiar decisões rápidas durante o atendimento  

---

## 🧪 Validação com o Cliente

A proposta foi construída com base em:

- Entendimento das dores reais dos usuários  
- Análise do fluxo atual  
- Identificação de gaps de usabilidade  
- Levantamento de hipóteses
